My past work

Here is a collection of the work that I have completed to date.

"The Pit" Remake:

In order to start learning Unreal Engine 4, I undertook the project of making a simplified version of "The Pit", from Call Of Duty: Modern Warfare 2. I did this as it allows me to implement some interesting features such as grenade explosions and a destructable environment while being able to use them in a simple game environment. I plan to take this project further, such as implementing a UI and better bullet physics before starting to add features which were not in the original version such as enemies AI's.

OpenGL - Solar System:

This was a University Project in which we had to learn the basics of OpenGL. This project also allowed for me to learn about scene graphs and implement them within my solar system. Making this was interesting as it was new teritory on multiple levels due to having never worked within a 3D environment, used a graphical API or implemented Scene Graphs before.


Splodey Guys:

This was the first game that I created and it was done using the Universities proprietary API called S2D. It is a basic two player clone of 'Bomberman' with various power ups dropped from the crates when they are designed. While basic, it intoduced me to the fundamentals of games programming and a better understanding of what goes into making a game.


Super Mario Bros:

As my second venture into games programming, this 'Super Mario Bros' clone introduced me to the SDL2 Library, game audio and also very basic enemy AI.
